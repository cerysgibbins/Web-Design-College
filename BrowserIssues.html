Below is the code which allows the button to work in all browsers. The reason why we use event.target.value is because it allows us to reference the target of the object. The target and srcElement both return the HTML element that the event took place on.

function reportMyBrowser(event) {

var eventButtonValue = "";
    if (window.event) {
        eventButtonValue = window.event.srcElement.value;
    }
    else {
        eventButtonValue = event.target.value;
    }
   
}

This code tells me that if the user is using Internet Explorer then the IF block of the code will work. The srcElement lives inside the window.event element. If the user isn't using Internet Explorer and they are using any kind of W3C web browsers including Firefox, Safari etc the bottom part of the code in the ELSE block will work. The event.target attribute targets the element that triggers the event. The event attribute is a string that specifies the event.
