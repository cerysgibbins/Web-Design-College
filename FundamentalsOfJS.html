What is JavaScript?

JavaScript is a client side scripting language and is known to add functionality and interactivity to HTML pages and much more. Javascript is also known to work well in other browsers, such as, Internet Explorer, Mozilla Firefox, Safari and others. Like CSS, there are many ways that JavaScript can be included in your HTML pages. It can be saved as an External file or included in the head section of your HTML pages. If you wanted to put your JavaScript code in the HTML code then you would need to use this link below:

<script type="text/javascript" src="script.js"></script>

What can it do?

There are many methods that can be used in JavasScript including, Variables, Functions, Events and many others. I will be explaining each of the methods and others below.

Variables

Variables are a way of holding together values and are created by using the var statement. Variables can hold text data, numerical data, true/false and others together. For the variable to work properly you will need to assign the = to the right hand side of the var statement. An example of a variable is below:

var myName = Cerys
var myAge = 16

Functions

A function is another method in JavasScript and performs tasks. The function will be executed by an event or you can give the function your own name. Functions are usually in the <head> or <body> section in the page. You will also need to make sure that when you call the function it gets loaded into the page before it is called. Click on the button below for it to alert you with a message box. For me to do this I used the function below the button: 

function displaymessage(){
alert("Hello, welcome to the JavaScript section of this Website.");
}

I don't know if you have noticed but I have also inserted a digital clock underneath the main menu to the right hand side, at the top of the page. I implemented this by using the scripting language tag and also using variables and functions . Below is the code that I used to do this:

<script type="text/javascript">
var timer = setTimeout(updateTheClock,1000);
							
function updateTheClock() {
var thisTime = new Date();
var h = thisTime.getHours();
var m = thisTime.getMinutes();
var s = thisTime.getSeconds();								
var time = h + ":" + m + ":" + s;
document.getElementById("myClock").innerHTML = time;
timer = setTimeout(updateTheClock,1000);
}
</script>

Events

An Event is another method that is used in JavaScript and they respond to actions that the user makes. For example, the button above under the functions section is an event because when you click on it a message is displayed to the user. I used the onClick button to make the message to display as when you click on the button the JavaScript code will be triggered and will be displayed. Below is the code that I used for the button: 

<input type="button" value="Click me!" onclick="displaymessage()" />

There are many different events that you can use, not just the onclick event. Another example of an event is onMouseOver, which can apply to an image or link when the user moves their mouse over the object. All of the event handlers within JavaScript all start with on and each of the events apply to different things. Some of the most common events that are used are, onClick, onChange, onLoad, onMouseOver and onKeyPress.

Overall, there are many things that JavaScript can do and one of the main things that it does is that it's a client side scripting language and it can change the page content dynamically.
